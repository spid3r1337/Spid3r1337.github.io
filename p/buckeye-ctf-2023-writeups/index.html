Starter Buffer | PWN

attachment: start-buffer.c

Tell me your favorite number and I might give you the flag ;).

netcat: nc chall.pwnoh.io 13372

I see the buffer overflow vulnerability here

so we can use pwntool to send byte 0x45454545 through netcat and string buffer and get the flag (The current value of the flag is 0xaabbccdd, so we must input 60 * "A" + byte 0x45454545 so that the value after input of the flag becomes 0x45454545)

p = remote("chall.pwnoh.io",  13372)

p.sendline(exploit_payload)
response = p.recvall()

import hashlib
from sys import exit

flag = '8f163b472e2164f66a5cd751098783f9'

str = input("Enter the flag\n")
str = hashlib.md5(str.encode())

if str.digest().hex() == flag:
    print("Congrats! You got the flag!")
else:
    print("Nope. Try again!")

exit()

it seems like an easy md5 reverse, use MD5 Center to reverse this MD5 hash, we get orchestra –> bctf{orchestra}
