import discord
from discord.ext import commands

Bot = commands.Bot(command_prefix="/",
                   description=description,
                   intents=intents)

@Bot.event
async def on_ready():
  print(f'Bot is ready. Log in as: {Bot.user.name}')

@Bot.command()
async def xinchao(ctx):
    await ctx.send('Hello!')

 Bot.run('YOUR_TOKEN')

Name this code main.py for the system that uses it

Note: YOUR_TOKEN is the Token you just received above

from flask import Flask
from threading import Thread

app = Flask('')

@app.route('/')
def home():
  return "ping me every time!!!"

def run():
  app.run(host='0.0.0.0', port=8080)

def keep_alive():
  t = Thread(target=run)
  t.start()

Name the code keep_alive.py along with the directory containing main.py

Host Bot on Replit

Let’s talk a little bit about Replit, this is a very good online IDE that supports many programming languages â€‹â€‹(I often use python here for the purpose of being able to install libs). specific to CTF without error). So if you can host a Bot on your computer, you can also host it on Replit

Hosting Bot is extremely easy by just putting the last 2 python files onto Replit and running it

After Run on Replit, you can run the Bot normally, but there is a disadvantage of Replit: after 5 minutes, if no user accesses and uses it, it will automatically shut down, that’s why we Need more tools to be able to ping Bot continuously

Keep Bot “awake” with Uptime RoBot

Uptime RoBot through the keep_alive.py file set up above helps check if the Bot is active and will ping it to replit to revive it so that it can be revived. Hang Bot 24/7 without fear of death.

Example with uptime statistics of my old Bot (Since 2710 hrs, 23 mins (2023-06-09 17:46:19))

Tutorial:

  1. Select + Monitor to create a new Monitor
  2. In Monitor Information –> Monitor Type select HTTP(s)
  3. Name and add the URL of Project Replit (it will look like https://.repl.co)
  4. Set Monitoring Interval to every 5 minutes (in accordance with Replit)
  5. Select Create Monitor and the setup is complete

Uptime RoBot will now have the task of pinging the Bot every 5 minutes, which is a lot of fun (unless you force it to do too much work with a free tool :3)
